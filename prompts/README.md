# Role Prompts

Each file in this directory is a **prompt** for one AI role in the team workflow. Use them when running the corresponding phase (see **WORKFLOW.md**).

| File | Role | Phase |
|------|------|--------|
| `manager.md` | Manager | 01 — Task analysis and decomposition |
| `researcher.md` | Researcher | 02 — Research and comparison |
| `architect.md` | Architect | 03 — System/solution design |
| `engineer.md` | Engineer | 04 — Implementation |
| `reviewer.md` | Reviewer | 05 — Validation and critique |
| `writer.md` | Writer | 06 — Final documentation |

Copy the prompt into your AI session along with the project context (brief, prior outputs). Write that role’s outputs into the project’s phase folder under `projects/<project-name>/<phase>/`.
