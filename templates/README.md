# Templates

Reusable document templates for the AI team workflow. Copy into the relevant project phase folder and fill in.

| Template | Used by | Phase folder |
|----------|---------|--------------|
| `research.md` | Researcher | `02-research/` |
| `architecture.md` | Architect | `03-architecture/` |
| `final-report.md` | Writer | `06-documentation/` |
| `knowledge-extraction.md` | Knowledge Extraction | `07-knowledge-extraction/` |

Manager, Engineer, and Reviewer do not have mandatory templates; use the structure in their role prompts instead.
