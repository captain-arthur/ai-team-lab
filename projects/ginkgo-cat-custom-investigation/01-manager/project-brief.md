# Ginkgo 기반 Custom CAT 조사 계획

**Date:** 2026-03-18  
**Program:** CAT

## 목적
- “Ginkgo로 완전히 커스텀한 CAT 시나리오(Go 기반)를 구현할 수 있는가?”를 **실험 가능한 수준의 계획**으로 정리한다.
- 동시에 “CAT 통합 오케스트레이션/플랫폼의 기반”까지 확장 가능한지 판단하기 위한 조사 범위와 평가 기준을 확정한다.

## 해결하려는 질문(핵심)
1. Ginkgo는 커스텀 CAT 구현 도구로 적합한가?
2. PASS/FAIL과 파일 결과 저장(기계 판정/증거)까지 가능한가?
3. k6/CL2가 맡는 역할과 충돌 없이 CAT에 편입/확장 가능한가?
4. 오케스트레이션 기반(플랫폼 레벨)까지 갈 수 있는가?

## 성공 기준
- 조사/실험/판단이 구분되어 있으며, 각 항목이 “무엇을 알아내기 위한 것인지(왜)”가 명확하다.
- 평가 축이 CAT 최소 조건(Scenario/SLI/SLO/Persistence)과 직접 연결된다.
- 다음 단계(실험/프로토타입 개발)로 바로 넘어갈 수 있을 만큼 구체적인 최소 실험 후보(2~3개)가 제안된다.
