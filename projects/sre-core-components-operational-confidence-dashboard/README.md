# Core Components Operational Confidence Dashboard

Kubernetes 운영 모니터링에서 운영자가 묻는 핵심 질문:
- "지금 안전한가?"
- "내일도 안전한가?"
- "지금 무엇을 해야 하는가?"

이 프로젝트는 `istio-ingressgateway`, `nodelocaldns`, `coredns` 3개 컴포넌트에 한정해, **operational confidence 중심**으로 Grafana 대시보드를 바로 설계/구현할 수 있는 문서(01~08)를 제공한다.

Foundation: projects/sre-monitoring-cluster-health-model
